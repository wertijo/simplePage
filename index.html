SISTEMAS DE INFORMACIÓN

Los sistemas de información están encargados de tomar ciertos datos, que al ser agrupados y estudiados, generan información, dicha información es recopilada, procesada, almacenada y difundida por medio del SI (sistema de información).
Para entender qué es un sistema de información se empezará definiendo qué es lo que se entiende por Sistema. Además, también se dará la definición de información para dejar más en claro que es un SI.

Sistema

El sistema es un conjunto de cosas, máquinas, personas, etc. Que se relacionan entre sí de forma racional con un propósito específico. Dicho proósito es la rezón de la existecia del Sistema.

Información vs Dato

Información

La información es el resultado del análisis de los datos, es decir que para obtener información es necesario primero tener datos, estos datos son procesados dentro de un contexto específico, lo que genera Información.
Además, si se procesa la información obtenida de los datos, se puede llegar a generar Conocimiento, entonces, basícamente se obtiene una utilidad de todo ese procedimiento. Estos procesos siempre van encaminados a una finalidad, como por ejemplo, tomar una decisión importante.

Para asegurarse que la información que se toma es valiosa o funcional, esta debe tener las siguient características:

  1. Completitud.
  2. Calidad.
  3. Precisión.

Dato

Como ya se vió, a partir del análisis de los datos se genera la información, de esta manera se puede decir que un datos es como materia prima. Esta materia prima, siempre se asocia en un contexto específico para ser analizada.
El datos, entonces, es la unidad más mínima de conocimiento que existe.

¿Qué es SI?

Entonces, ya vistos los conceptos de información y de sistema, un sistema de información es un conjunto de procesos que se aplican a ciertos datos, lo que genera información y esta misma información es la que se recopila en el sistema. Este proceso de recolección de información se hace con una finalidad previamente definida por la entidad responsable del SI.

Los elementos de in Sistema de información son:

  1. Recursos físicos
  2. Personal
  3. Normas
